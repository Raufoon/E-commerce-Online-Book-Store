    Project maintained by Raufoon Hosted on GitHub Pages — Theme by mattgraham

    Project Title: Arunachol.com - An Online Website for buying and selling secondhand books

    Course: CSE 4114- E Commerce & Web Engineering Lab

    Platform

    • Server: Apache
    • Database: MySQL
    • Language: PHP, JS, CSS
    • Framework: Laravel 5.2

    Features

    • Users' advertisement of books
    • Buying and Selling
    • Using cart to shop books
    • Conducting business online
    • Payment by online and on delivery
    • Employees of website can work online on their job
    • Profile Management
    • Security

    Views

    • Index page
    • Profile page
    • My ads page
    • Purchase history
    • Search results
    • All ads
    • Manager's office page
    • Manager's employee list
    • Order to take book from seller
    • Order to deliver books to buyer
    • Cash memo
    • Cart: (a floating cart which can be accessed from all pages)

    Limitations

    • Auction system not implemented
    • Layout not flexible
    • Ajax requests are not efficiently handled
    • Small features like- upvoting and downvoting, messaging, following etc aren't implemented

    Future plan

    • Introduction of auction system
    • More security
    • Flexible UI
